Тестовая задача на вакансию php-программист.

Необходимо написать js-скрипт, который работает с API системы amoCRM. Скрипт должен для всех контактов без сделок создать новую задачу с текстом “Контакт без сделок”.

Инструкция запуска: 
1.  `npm init`,
2.  `npm install amocrm-js`,
3. Подставить нужные данные интеграции в сприпт,
4. запустить с помощью команды `node index.js`.
